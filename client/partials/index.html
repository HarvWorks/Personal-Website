Portfolio

Aarocare

Aarocare is a cross platform app that allows rural Indians to request a certified local doctor or nurse to visit their house for small treatments such as vaccines, general sickness, triage care, etc. First, this grants the rural Indian population a cheaper access to healthcare due to the fact that they no longer have to take a costly taxi or train to the city center. Seconds, this allows doctors and nurses who live in the rural area an additional income source that they otherwise would not have.

The completed app will consist of both a patient app and a caregiver app on both the Android and iOS plaforms. Additionally a website is made available for doctors and adminstrators.


Technologies Used:

Xcode 8, Swift, Android Development Studio, Java, JSON Web Tokens (JWT), Socket.io, CoreData, Apple Maps API, Google Maps API, Node, Java Script, MySQL, Bootstrap, Express, Bower, Bootstrap

MyFace 2.0

MyFace 2.0 is a Facebook clone with a twist. Like Facebook, you can friend people, unfriend people, block people, post on a Wall, edit your post, delete it, and most importantly chat with people in real time. However, unlike facebook, the chat window is also a map that serves as a way to plan evening trips, and daily events!


Technologies Used:

Flask (refined into a framework known as Pylot), JavaScript, Google maps API, Socket.io, MySQL, Bootstrap

Pop It

Pop It is a modern take on the tried and true game of Bop It. It ultizes noise reduction formulas and motion control to allow the game to detect 9 distinct types of motions! Also it contains some very unique sounds.


Technologies Used:

Xcode 8, Swift, CoreMotion, Core Data, AVFoundation

New Bullet Hell

New Bullet Hell is a top down shoot-em-up built in mostly pure JavaScript with some help from Paper.js. The game also contains no images, everything is drawn using JavaScript.
